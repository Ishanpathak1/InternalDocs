For each new worker, the following should be done:
1. Obtain office space.  Clean the desk area.
2. Ensure a computer and monitor(s) are available.
    a) If a computer is not available, determine the budget your project has available and ask the IT manager to order one.
    b) specify any special software the worker needs, other than Microsoft Office.
3. As soon as the worker has a login id, contact IT manager to add them to the CHSR Research Group
4. As soon as the worker has an email, contact IT manager to add them to the ListServe for CHSR
5. When the computer is ready, and they have logged in, ask CHSR IT to set them up with the CHSR Default printer and access to G: drive
6. Walk new employee around to meet everyone.
