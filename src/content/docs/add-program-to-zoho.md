1. Go to Settings Gear Control, next to your Zoho picture, in the top right corner.
2. Under Customization, click on Layouts &amp; Fields
3. Click on CHSR and a &#39;Ticket Information&#39; layout should appear, starting with Contact Name at the top.
4. Find Program, hover your cursor over the field, and a gear icon should appear.  Click it.
5. Choose &quot;Edit Properties&quot;
6. Hover over an item in the &quot;Pick List Values&quot; and click the + icon that appears.
7. Type in the Product Name &lt;space&gt; ProgramPK &lt;space&gt; Program Name
NOTE: Any special character will fail (commas, pound signs, dollar, etc.)
NOTE 2: No special characters can be in the application program table either, as this is sent to Zoho and matches on the name you added in step 7.
