G Drive:

1. Add a network drive that points to \\lincoln.univ.albany.edu\SSW\CHSR_Research.

Konica:

1. Go to G:\DDISS\KonicaDriver.

2. Copy the entire Win_x64 folder to your computer

3. Go to Printers &amp; scanners and click the &quot;Add device&quot; button

4. Wait while the list populates until you see the &quot;The printer that I want isn&#39;t listed&quot; line.  Click the &quot;Add manually&quot; link.

5. Choose the &quot;Add a printer using an IP address or hostname&quot; option and click Next.

6. Change the device type to &quot;Autodetect&quot;, enter 10.200.104.11 as the IP address, and ignore the port name field.  Uncheck the &quot;Query the printer and automatically select the driver to use&quot; checkbox.

7. Click the &quot;Have Disk&quot; option, click the &quot;Browse...&quot; button, and select the KOAYQW_.INF file that is contained in the Win_x64 file you copied in step #2.  Click &quot;OK&quot;.

8. That should be it!  Print a test sheet to ensure that it worked.
